Kde Ubuntu 18.04安装笔记

2019-10-30

安装设置

  • 语言选择英语，避免/home出现中文文件夹。
  • 选择最小安装；勾选为图形或无线硬件，以及其它媒体格式安装第三方软件。
  • 双系统安装，无需设置/boot分区，只许把选择把引导安装在Windows Boot Manager即EFI分区即可。
  • 其它选项按字面意思常规处理即可。

安装xrdp服务

这真是个大坑！要特别注意安装包的顺序问题。

  1. 安装软件包。

    #先安装这个
sudo apt-get install tightvncserver
#再安装这个
sudo apt-get install xrdp、
#再重启一下xrdp服务
sudo /etc/init.d/xrdp restart

  2. 根据桌面环境进行设置。

    根据具体的桌面环境，还需进行相应的操作。推荐从这个网站的Remote desktop server一项中根据自己的桌面环境进行选择。

  3. xrdp无法成功开启的解决办法。

    查看/var/log/xrdp-sesman.log文档：

    sudo nano /var/log/xrdp-sesman.log

    如果文档中有类似X server for display 10 startup timeout和another Xserver is already active on display 10类似的日志，很大可能是安装包顺序的问题，解决办法就是卸了重新按顺序安装。

    apt-get purge tightvnc xrdp

    然后按照第一步的顺序安装。也可只卸载重装xrdp包（反正我是这么解决的）。

    这是我遇到的唯一问题！

安装Petalinux(Xilinx)

注意：不能使用root权限来安装。

  1. 报错awk: read error (Bad address),参考这篇,只许安装gawk:

    sudo apt install gawk

  2. 报错tar: .: Cannot change mode to rwxr-xr-x: Operation not permitted这种类似错误，参考这里，需要修改文件的拥有用户：

    #例如安装在/opt/Xilinx/Petalinux/2018.3,<owner>为非root用户名，可使用echo $USER查看
sudo chown -R <owner>:<owner> /opt/Xilinx/Petalinux/2018.3
