数字电路时序初步及最小时钟周期计算

2019-07-12

几个时间概念

TclkT_{clk}Tclk​:时钟周期。两个时钟上升沿之间的时间。

TcoT_{co}Tco​ / TctoqT_{ctoq}Tctoq​:寄存器更新延迟。clock output delay，时钟触发到数据输出的最大延迟时间。

TgateT_{gate}Tgate​ / TdataT_{data}Tdata​:逻辑门的延迟。一般包括传播延迟。

TsetupT_{setup}Tsetup​:触发器的建立时间。

TholdT_{hold}Thold​:触发器的保持时间。

TskewT_{skew}Tskew​:时钟偏移。同一时钟到达不同的触发器的时钟引脚经历的路径可能存在差异，造成他们的时钟上升沿不是同时出现的，这种偏差称为时钟偏移。通过使用时钟树综合工具可以有效地减少时钟偏移，但是不能消除时钟偏移。
